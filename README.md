# SASDT: Semantic-Aware Transformation Discovery for Heterogeneous Data

## Installation

1. Install Python 3.12.5
2. Install libraries with `pip install -r requirements.txt`
3. Install Ollama and download Qwen3-4B, Llama3.2-3B-Instruct, and Phi4-mini
4. OpenAI API required

## Datasets

* `data/`: original data
* `data_shuffled/`: shuffled data
* `data_perfect/`: ground truth for each key component
